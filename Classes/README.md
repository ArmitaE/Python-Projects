# [King Arthur](./King_Arthur.py)

This project shows example of inheritance in python by printing the story.

```
Smaug created.
-------------
Smaug hp = 300.
Smaug attack = 10.
Smaug acid = 20.
-------------
Arthur created.
Galahad created.
-------------
Arthur hp = 50.
Arthur attack = 5.
Arthur potion = 20.
Galahad hp = 20.
Galahad attack = 5.
Galahad potion = 20.
-------------
Smaug attacks the Arthur and deals 10 damage.
Smaug attacks the Galahad and deals 10 damage.
-------------
Arthur hp = 40.
Arthur attack = 5.
Arthur potion = 20.
Galahad hp = 10.
Galahad attack = 5.
Galahad potion = 20.
-------------
Smaug burns the Arthur and deals 20 damage.
-------------
Arthur hp = 20.
Arthur attack = 5.
Arthur potion = 20.
-------------
Galahad uses potion to heal the Arthur and heals 20 hp.
-------------
Arthur hp = 40.
Arthur attack = 5.
Arthur potion = 20.

```
